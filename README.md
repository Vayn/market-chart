# 一个基于React和百度Echarts的行情K线图和分时图展示应用

## 安装
```$ git clone <repository-url>```
```$ npm install```

## 运行
```$ npm start```

## 访问
在浏览器输入 `http://localhost:8080/#/TJAG` 进行访问
